# Scorekeep

Simple web app for all score keeping purposes.

## Setup

### ```
git clone https://github.com/cherrynoize/scorekeep
cd scorekeep
npm i
```

## Usage

### `npm run start`

Go to [http://localhost:3000](http://localhost:3000) to view the
app in your browser.

### `npm run build`

Builds the app for production to the `build` folder.
